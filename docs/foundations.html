1.3 Geographic Data in R

Note

A few core libraries underpin spatial libraries in R (and Python!) and in software applications(QGIS, ArcPro) and spatial data structures across languages and applications is primarily organized through OSgeo and OGC). These core libraries include: - PROJ –> Spatial projections, transformations

  • GEOS –> Geometry operations (measures, relations)

  • GDAL –> Raster and feature abstraction and processing (read, write)

  • NetCDF –> Multidimensional (XYZT) data abstraction (read, write)
